👑 1. Tutorial ADMIN

Tugas Utama:
Mengelola master data, pengguna, dan pengaturan sistem.


(Tekan panah Bawah (↓) untuk detail Admin)

Login & Dashboard

  • Login: Gunakan email admin@sias.com
  • Dashboard: Pantau statistik sekolah seperti Jumlah Siswa, Pegawai, Kelas, dan Ekskul.

Manajemen Pengguna

  • Buka menu Master Data > Manajemen User
  • Buat akun baru secara manual atau setujui (approve) registrasi user baru.
  • Tetapkan Role (Hak Akses) kepada masing-masing pengguna agar mereka bisa mengakses menu yang sesuai.

Master Data & Backup

  • Manajemen Kelas: Buka menu Kesiswaan > Data Kelas untuk membuat kelas.
  • Tahun Ajaran: Atur Tahun Ajaran dan Semester aktif di menu Akademik.
  • Backup: Buka Pengaturan Website > Backup & Restore. Klik "Buat Backup Baru" secara rutin.

👨‍🏫 2. Tutorial GURU

Tugas Utama:
Mengecek jadwal, piket, dan ekskul.


(Tekan panah Bawah (↓) untuk detail Guru)

Jadwal Mengajar

  • Dashboard: Akan langsung menampilkan Jadwal Mengajar Hari Ini secara otomatis.
  • Buka menu Akademik > Jadwal Mengajar untuk melihat seluruh jadwal dalam seminggu.
  • Anda bisa mengunduh jadwal ini dalam format PDF.

Piket Harian (Jika Bertugas)

  • Buka menu Piket > Dashboard Piket
  • Isi Catatan Harian Piket tentang kejadian hari itu.
  • Catat Izin Keluar jika ada Siswa/Pegawai yang keluar sekolah.
  • Input Absensi Manual (Sakit/Izin/Alpha).

Pembina Ekskul

  • Buka menu Ekstrakurikuler > Data Ekstrakurikuler
  • Kelola daftar anggota ekskul Anda.
  • Isi absensi kehadiran anggota pada setiap pertemuan.
  • Catat prestasi atau kejuaraan yang diraih.

🧑‍⚕️ 3. Tutorial GURU BK

Tugas Utama:
Memantau kedisiplinan dan kasus siswa.


(Tekan panah Bawah (↓) untuk detail BK)

Jurnal & Kedisiplinan

  • Jurnal BK: Catat aktivitas harian BK di menu Bimbingan Konseling > Jurnal Harian BK.
  • Kedisiplinan: Input data Pelanggaran Siswa beserta bobot poinnya.
  • Catat juga data kedisiplinan Siswa Terlambat.

Penanganan Kasus

  • Panggilan Ortu: Buat surat/catatan pemanggilan jika poin pelanggaran siswa sudah melampaui batas (Menu Alih Tangan Kasus).
  • Alih Tangan (Referral): Rujuk masalah atau kasus berat ke Kepala Sekolah atau psikolog melalui sistem.

👨‍🎓 4. Tutorial SISWA

Tugas Utama:
Mengakses informasi akademik.

Fitur untuk Siswa

  • Login: Menggunakan NISN/Email dan Password yang diberikan Admin.
  • Dashboard: Melihat ringkasan informasi, pengumuman sekolah, dan jadwal.
  • Blog & Berita: Membaca artikel dan pengumuman terbaru dari sekolah.

Catatan: Akses siswa bersifat read-only (hanya melihat) untuk informasi publik dan data akademik pribadinya.

💬 Fitur Global: Chat Internal

  • Semua role (Admin, Guru, BK, Siswa) dapat menggunakan fitur Chat di sidebar.
  • Mulai pesan pribadi (1-on-1) dengan guru atau teman.
  • Buat Grup Chat untuk koordinasi kelas atau tim ekskul.
  • Mendukung pengiriman pesan teks, gambar, video, dan dokumen.
